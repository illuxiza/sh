1. 运行 
```
bash <(curl -fsSL https://github.com/go-gost/gost/raw/master/install.sh) --install 
```
安装gost
2. 运行 
```
bash <(curl -fsSL https://github.com/illuxiza/sh/raw/main/bggs/gen-gost-s5.sh) 
```
进入生成代理可视化界面

根据菜单生成对应的代理

3. (可选)
运行 
```
bash <(curl -fsSL https://github.com/illuxiza/sh/raw/main/bggs/gen-gost-s5.sh) -l
```
直接显示所有配置


运行 
```
bash <(curl -fsSL https://github.com/illuxiza/sh/raw/main/bggs/gen-gost-s5.sh) -s id
```
直接启动对应id的代理

运行 
```
bash <(curl -fsSL https://github.com/illuxiza/sh/raw/main/bggs/gen-gost-s5.sh) -k id
```
直接关闭对应id的代理

运行 
```
bash <(curl -fsSL https://github.com/illuxiza/sh/raw/main/bggs/gen-gost-s5.sh) -d id
```
直接删除对应id的代理
